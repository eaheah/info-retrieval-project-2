# info-retrieval-group2-project1

## Set up
- Install python 3
- create a virtualenv with python3 (`python3 -m venv /path/to/venv`)
- `source /path/to/venv/bin/activate`
- `pip3 install --upgrade pip`
- `pip3 install -r requirements.txt`
